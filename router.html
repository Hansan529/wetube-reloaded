#4.0 What are Routers?

기본적인 URL 경로이다.

      / → home
      /join → Join
      /login → Login
      /search → Search

      /edit-user → Edit User
      /delete-user → Delete User
      /watch-video → What Video
      /edit-video → Edit Video
      /delete-video → Delete Video

해당 주제에 대해 라우터를 통해 정리해겠다.

라우터는 작업중인 주제를 기반으로 URL을 그룹화한다.

      // /edit-user → Edit User
      /users/edit → Edit 

      // /delete-user → Delete User
      /users/delete

      // /watch-video → What Video
      /videos/watch

      // /edit-video → Edit Video
      /videos/edit

      // /delete-video → Delete Video
      /videos/delete
